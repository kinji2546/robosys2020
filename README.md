#Σの計算ぷろぐらむですが詳しく説明  
*このプログラムは、連続する数を足し合わせるものです。例えば、1から5までの和は1 + 2 + 3 + 4 + 5であり、これは15です。  
脱線しますが、1からnまでの自然数の和は「n(n + 1) / 2」で計算できます。

#実行例

`$ seq 6 | ./plus`
